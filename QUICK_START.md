# 快速开始指南 - V2.0 增强点位系统

## ⚡ 5分钟快速上手

### 第一步：启动应用
```
1. 打开应用
2. 看到罗盘和"堪舆案例"选区
3. 已默认创建"样本案例 1"
```

### 第二步：创建自己的案例
```
1. 点击绿色按钮"+ 新建案例"
2. 输入案例名称，例如: "办公室风水分析"
3. 点击"创建" ✓
4. 新案例自动成为当前活跃案例
```

### 第三步：添加第一个原点
```
1. 调整手机，对准想要分析的位置
2. 观察罗盘中央的红色十字准星
3. 当准心对准目标时，点击"+ 在十字准星处加点"
4. 在弹窗中:
   - 选择类型: 🔴 原点 (已默认)
   - 选择案例: 办公室风水分析
   - 输入名称: 例如 "办公室"
5. 点击"保存" ✓
```

### 第四步：添加终点（自动生成连线）
```
1. 再次调整手机，对准另一个目标位置
2. 点击"+ 在十字准星处加点"
3. 在弹窗中:
   - 选择类型: 🔵 终点 (需要切换)
   - 选择案例: 办公室风水分析
   - 输入名称: 例如 "窗户"
4. 点击"保存" ✓
5. **系统自动生成连线!** 🎉
```

### 第五步：查看分析结果
```
方法一：查看点位详情
- 点击"📍点位(2)"
- 查看两个点的详细信息（方位/五行/八卦等）

方法二：查看连线信息 ⭐
- 点击"📈连线(1)"
- 查看原点→终点的方位角、距离、山位等专业风水信息
```

## 🎯 常用操作速查表

### 高频操作

| 操作 | 步骤 | 说明 |
|------|------|------|
| 新建案例 | 点击"+ 新建案例" | 创建独立的分析项目 |
| 切换案例 | 点击案例标签 | 快速切换工作案例 |
| 删除案例 | 长按案例标签 | 删除整个项目（谨慎操作） |
| 加点 | 点击"+ 在十字准星处加点" | 添加原点或终点 |
| 查看点位 | 点击"📍点位(N)" | 列打开点位详情 |
| 查看连线 | 点击"📈连线(N)" | 显示所有连线信息 |
| 删除点 | 在点位列表中点击[删除] | 移除单个点位 |
| 删除连线 | 在连线列表中点击[删除] | 移除单条连线 |

### 加点时的选择

**🔴 原点 (Origin)**
- 参考点/基准点
- 例如：住宅、办公室、店面等
- 最多 2 个/案例

**🔵 终点 (Endpoint)**
- 分析目标
- 例如：窗户、门、财位等
- 最多 5 个/案例

## 💡 使用技巧

### 技巧 1：精确定位
```
使用十字准星的红点（中心）而不是整个十字
- 确保手机水平放置
- 缓慢旋转找准目标
- 看到准心对中时再保存
```

### 技巧 2：案例分组
```
不同项目用不同案例隔离:
- "客户A工厂" - 原点：工厂, 终点：车间/仓库
- "客户B住宅" - 原点：住宅, 终点：卧室/书房
```

### 技巧 3：批量分析
```
一个原点 + 多个终点:
- 添加1个原点（例如：办公室）
- 依次添加5个终点（例如：5个座位）
- 自动生成5条连线进行比对
```

### 技巧 4：历史对比
```
保留多个案例以历史对比:
- "项目A v1" - 初始分析
- "项目A v2" - 改进方案
- 对比两个方案的连线信息
```

## 🎨 界面说明

### 主要按钮

```
┌─ 绿色 "+"        └─ 新建案例
┌─ 蓝色 "📍点位(N)" └─ 查看点位列表  
┌─ 橙色 "📈连线(N)" └─ 查看连线列表
┌─ 红色 "+"        └─ 在十字准星处加点
┌─ 深蓝 "🧭"       └─ 校准罗盘
```

### 统计信息

```
┌──────────────────┬──────────────┬──────────────┐
│  原点数: 1       │  终点数: 2   │  连线数: 2   │
└──────────────────┴──────────────┴──────────────┘
- 原点数：当前案例有多少个原点
- 终点数：当前案例有多少个终点  
- 连线数：自动生成的原点-终点连线总数
```

### 罗盘信息

```
中央十字准星 (红色 #)
  │
  ├─ 角度显示：当前指向角度 (0°-360°)
  ├─ 山位显示：对应的 24 山 (壬、子、癸...等)
  └─ 分金显示：对应的分金信息 (甲子、丙子...等)
```

## ⚠️ 注意事项

### 必读

- ✓ **保存数据**：应用重启会丢失数据（后续支持自动保存）
- ✓ **校准罗盘**：首次使用强烈建议点击"🧭 校准罗盘"
- ✓ **手机方向**：需水平放置，避免倾斜导致指向不准
- ✓ **金属干扰**：远离金属物品，避免磁场干扰

### 常见误区

❌ 误区：十字准星越大越好  
✅ 正确：使用红点中心精确定位

❌ 误区：边走动边加点  
✅ 正确：固定位置对准后再保存

❌ 误区：一个案例用很久  
✅ 正确：不同项目建不同案例

## 🔄 工作流程示例

### 场景：分析公寓的最佳摆放位置

```
第1步 创建案例
  → "小张公寓家具摆放"

第2步 添加原点 (参考点)
  → 选择客厅中心 → 标记为 "客厅中心"
  
第3步 添加终点 (备选位置)
  → 选择沙发位置1 → 标记为 "沙发A"
  → 选择沙发位置2 → 标记为 "沙发B"  
  → 选择书桌位置   → 标记为 "书桌"
  
  [系统自动生成3条连线]

第4步 分析结果
  → 查看3个终点相对客厅中心的方位
  → 根据五行、八卦选择最优摆放
  
结论：
  沙发A (东南) → 木属性 → 利于财运
  书桌 (北) → 水属性 → 利于学习
```
