# fengshui-tool

V0: Android FengShui compass prototype (map abstraction, compass overlay, single-case point flows, persistence, trial-limit).

Quick start

1. Open in Android Studio.
2. Build with Gradle: `./gradlew assembleDebug`.

Notes
- Map providers under `app/src/main/java/com/fengshui/app/map/abstraction` are placeholders; integrate real SDKs for full behavior.
- Registration sample code: valid demo code `TRIAL-UNLOCK-2026`.
